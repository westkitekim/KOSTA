//실제 웹브라우저가 웹의 클라이언트 프로그램 

HTML : HyperText Markup Language
	   웹 문서의 컨텐트를 정의하는 언어 
	   
	   웹브라우저에 의해 서버에서 전송된 html 정보를 입력받아 번역해서 웹 화면을 구성하게 한다
	   --> 서버는 out.print 해서 출력하는데 html 텍스트를 출력하고 브라우저는 번역하여 화면으로 구성
	   
CSS : 웹 문서의 디자인 스타일 및 레이아웃을 담당(웹 디자인 담당)

JavaScript : 웹 문서의 행위를 담당(자바스크립트의 기본 역할)	

   

XML : 주로 설정 정보를 명시할 때 사용. 확장형 마크업 언어
