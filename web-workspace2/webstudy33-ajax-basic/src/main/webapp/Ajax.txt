** Ajax **

▣ Asynchronous Javascript And Xml
  자바스크립트 비동기 통신
  
▣ Ajax 
: 자바스크립트 비동기 통신을 의미
  Ajax 방식은 필요한 ★데이터만 응답받아 처리하는 방식★이다
  (기존의 웹 통신 방식은 웹 페이지를 응답하는 방식이다)
		 
  비동기 통신 -> 요청 후 자신의 작업을 수행하고 서버가 응답할 경우 응답정보를 반영
  동기 통신 -> 요청 후 응답할 때까지 대기
		 
  Ajax 안에서도 비동기/동기 구분을 할 수 있다. 하지만 Ajax는 기본적으로 비동기 방식임
		 
▣ XML 과 JSON 
: Ajax 에서 데이터 통신용으로 사용됨
  구조화된 데이터를 송/수신하기 위해 사용한다
  초반에는 XML이 사용되었으나 최근에는 JSON이 주로 사용된다
			   
▣ JSON ( Javascript Object Notation ) 
: 경량의 데이터 교환 형식(XML 은 무겁다)
  자바스크립트 객체 표기법으로 작성된 일반 텍스트
